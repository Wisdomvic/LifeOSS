## Description
LifeOS is a web application that brings together 
our daily focus task, habit tracker and mood logger 
into one place, so that users no longer need multiple 
apps to manage their daily life. It runs entirely in 
the browser with no login or sign up page required.

## Features
- 💡 Daily Spark — where daily motivation helps 
   users to keep going and pushing

- 🌤 Weather — used to check the condition of 
   the weather before acting daily

- 🎯 Daily Focus — where you put down the task 
   you want to focus on

- ⏱ Pomodoro Timer — helps you stay focused on 
   your 25 minute timed work sessions and 
   automatic break reminders

- ✅ Habit Tracker — where you track or add the 
   daily habits that improve your personal growth

- 📝 Quick Notes — where you jot down important 
   information which automatically saves

- 🌊 Mood Logger — where you can choose your mood 
   and see a 7 day chart of your mood history

- 📅 Countdowns — counts down the days until an 
   upcoming event which you set


   ## Technologies Used
- HTML — creates the structure and layout of 
  the web application, every button and section 
  you see is created by HTML

- CSS — contains all the styling used to 
  beautify the webpage

- JavaScript — makes everything interactive 
  and handles saving data and fetching 
  live weather

- Open-Meteo API — used to fetch live weather 
  data based on the user's location

- Nominatim API — used to convert location 
  coordinates into a readable city name
